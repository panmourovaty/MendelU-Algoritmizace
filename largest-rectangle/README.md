# Largest Rectangle
## Task
Given pairs of numbers representing the sides of rectangles, print the sides of the three rectangles with the largest area to standard output.

## Expected result
Expected result of this code with demodata.txt should be **10.010.0 9.09.0 8.010.0**
```
./main < demodata.txt 
10.010.0
9.09.0
8.010.0
```
