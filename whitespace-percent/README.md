# WhiteSpace Percent
## Task
Write a complete program in C++ that solves the following task:

The input consists of a **sequence of characters** (we do not know their number or the last character). Print the **percentage** of all input characters that are **whitespace characters**: space, tab (\t), newline (\n).

## Expected result
Expected result of this code with demodata.md should be **10.1358 %**
```
./main < demodata.md 
10.1358
```
